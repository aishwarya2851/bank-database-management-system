This is a bank account management system whih helps both customers and employee of the bank to maintain the account database.
 This system maintains  Accounts information , Withdrawal, Deposit amount. The Database has been designed for reporting data such as  monthly statements, transaction details of various account types like checking, savings account and loan . So that it reduces time consumption, no paper work required . 
This Database is designed and built on MySQL platform.The project is carried out using PHP as front end and oracle based software as back end.
